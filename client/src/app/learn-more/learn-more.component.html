Expense Tracker is the place where users can track their cryptocurrency purchase history.

Registration

To be able to use this app, the user has to register first.

Login

If you have successfully registered, you can log in to use the app.

Expenses

The main use of this app is to track your expenses, i.e. your purchase history.
Every entry is defined with a cryptocurrency (Asset), date of purchase, and amount.
Cryptocurrency is chosen from the list of available currencies (Assets).
Date of purchase is entered using a date picker, and the amount is entered using a number input control.
Below the input form, users can see the table with all their inputs.
Users can remove an entry by clicking on 'X' button in the table row.

Assets

Assets are used for the dropdown control on the Expenses tab, for users to select an Asset from the list of available ones.
Assets are managed by Admins - only Admins have the rights to add or remove currencies.
Regular users will not see the Asset tab.

Admin Panel

This application has 3 kinds of roles: Super Admins, Admins, and Regular users.
Super Admin is the top-level role & they can use this panel to manage roles for users.
Regular users will not see the Admin tab.

Errors

Error tab demonstrates how different kinds of errors will display in the application.
This application is used for practice purposes, the Error tab would not be displayed for production of course.
